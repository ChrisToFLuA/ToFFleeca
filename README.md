# Jewels Robbery
- Break vetrin using qtarget function
- Alarm goes ON
- Search in the office, the code to stop the alarm
- Input the code on the Box
- Stop the alarm allow you to loot more jewels
- 10 vetrins in the jewelry store are breakable
